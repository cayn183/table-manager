# Release v0.6.3 - UI/UX Improvements

## What's New

### Added
- **Improved View Toggle Controls**: Enhanced "Kartenansicht" (Map) and "Planansicht" (Timeline) buttons with better styling and frosted glass effect in header
- **Event Save Button Reorganization**: Moved Event speichern button to sidebar footer with compact design
- **Compact Status Indicators**: Auto-save countdown (MM:SS format) and last save timestamp in compact labels (11px)
- **Print Button Integration**: Print button now integrated into event save section

### Changed
- **Layout Restructuring**: Moved view controls from floating sidebar to main header
- **Visual Hierarchy**: Enhanced toggle button styling with better borders and shadow effects
- **Form Factor**: Removed floating sidebar elements, integrated controls into main UI
- **Status Display**: Compact status labels for auto-save and last save information

### Fixed
- Removed duplicate "Event speichern" button and status displays from header
- Cleaned up floating sidebar UI elements
- Improved overall UI/UX consistency

## UI Improvements Summary

The Room component (`/room` page) has been completely redesigned with a focus on better UX:

1. **Header Controls**: View toggle buttons now integrate seamlessly into the header with improved visual design
2. **Sidebar Reorganization**: Event save controls moved from header to sidebar footer with auto-save status
3. **Cleaner Layout**: Removed visual clutter from floating sidebars
4. **Better Visual Feedback**: Improved button states and transitions
