# Table-Manager

This repo contains a minimal frontend scaffold for a browser-based table assignment app.

Quick start (after installing dependencies):

```bash
npm install
npm run dev
```

The app is a Vite + React (TypeScript) single-page frontend. CSV import uses PapaParse.

Notes:
- Import a CSV with a header (columns like `family,count` or `group,size`) or paste CSV text.
- Use "Auto Assign (Best-Fit)" to distribute groups across the defined tables.
